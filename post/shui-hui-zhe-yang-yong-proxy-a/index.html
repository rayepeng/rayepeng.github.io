谁会这样用Proxy啊🤣

2023-09-19 1 min read # js 
class T{
    constructor(){
        this.map = new Map()
        this.map.set(this, 10)
    }
    getvalue(){
        console.info(this.map.get(this))
    }
}

let a = new T()

let b = new Proxy(a, {})

b.getvalue()

原因：Proxy对象内部的this和原始对象内部的this指向不一样

为了表现出这个特性，于是我们在函数构造的时候就把当前的this存下来

对于内置类，Date内置类的强依赖this导致不能代理

const target = new Date();
const handler = {};
const proxy = new Proxy(target, handler);

proxy.getDate();
// TypeError: this is not a Date object.
