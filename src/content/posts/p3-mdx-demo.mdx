This page exists to **verify P3** of the astrofu migration plan:
the Markdown pipeline has been upgraded to MDX, headings get clickable `#`
anchors via `rehype-autolink-headings` (`behavior: 'append'`), code blocks
carry the `.shiki` class so the antfu.me `markdown.css` rules light up,
and MDX components hydrate as Vue islands.
